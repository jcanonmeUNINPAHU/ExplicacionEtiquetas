Elementos Principales

Elemento : < !DOCTYPE html >

Todos los documentos HTML deben comenzar con una declaración, La declaración no es una etiqueta HTML. Es una "información" para el navegador sobre qué tipo de documento esperar.

Elemento : < html >

El elemento es el elemento raíz de una página HTML.

Elemento : < Head >

conocida en español como cabecera, es la parte que no se muestra en el navegador en el momento de cargar la página. Esta etiqueta puede contener varios hijos que otorgan información sobre el sitio en cuestión.

Elemento : < body >

La etiqueta HTLM < body> se utiliza para definir el cuerpo del documento HTML. Específicamente, es la parte en donde se describe el contenido de la página, colores, forma, estructura, texto, imágenes.
Para indicar la etiqueta, se inicia con < body> y se culmina con < /body>, por lo tanto, el uso de ambas es obligatorio.

Elemento : < title >

El elemento especifica un título para la página HTML (que se muestra en la barra de título del navegador o en la pestaña de la página)

Elementos Estructurales

Elemento : < header >

es normalmente utilizada como el encabezado de una sección, en ellos pueden encontrarse diversos elementos como vínculos, logos, tablas de contenido o algunas introducciones.

Elemento : < article >

Permite separar diversos elementos independientes que puedan estar contenidos en un mismo < section >. Puede ser un mensaje de foro, un comentario, un artículo o una entrada en un blog, que permite ser reutilizada o distribuible.

Elemento : < section >

Representa una sección genérica de un documento. Sirve para determinar qué contenido corresponde a qué parte de un esquema. Piensa en el esquema como en el índice de contenido de un libro; un tema común y subsecciones relacionadas.

Elemento : < aside >

Podemos verlo como la representación de una sección, en la cual hay información que está relacionada indirectamente con el contenido principal descrito en el documento.

Elemento : < nav >

Es simplemente un contenedor de enlaces de navegación. Esta etiqueta te permite dirigirte a otras secciones o a enlaces externos. Es muy útil en la organización del contenido de la página.

Elemento : < footer >

Podemos definirlo de una manera sencilla como el pie de una sección, un pie de página, que es muy probable que contenga información acerca del autor o creador de dicha sección o datos, como los derechos de autor. Es utilizado comúnmente para colocar firmas, información de licencias, los cuales pueden encontrarse consecutivamente a lo largo del sitio.

Elemento : < div >

Con la etiqueta div (que se refiere a división), puedes agrupar los diferentes contenidos o hacer secciones. Es obligatoria y se escribe < div > y < /div >. En ella puedes encontrar texto o elementos en línea o bloques.

Elementos Visibles

Elemento : < ol >

Es un elemento de bloque que nos permite hacer listas ordenadas alfanuméricamente generadas de forma automática a medida que escribimos diferentes puntos de información, representados a su vez con la etiqueta < li >.

El código correcto requiere la etiqueta < ol> al inicio. Después hemos de colocar cada elemento del listado precedido por la etiqueta < li> y delimitado por < /li>. Cuando el contenido esté completo, se cierra con la etiqueta < /ol>.Una lista ordenada puede contener texto y otros elementos como tablas, imágenes e incluso sublistas (listas anidadas) tanto ordenadas como desordenadas.
Ejemplo:

  1. Enero
  2. Febrero
  3. Marzo

Tipo:
Permite definir el tipo de identificador que aplicaremos para dar orden a la lista. Puede tomar distintos valores: “1” – Orden lógico numérico, por defecto. “a” o “A” – Orden alfabético en minúsculas o mayúsculas. “i” o “I”– Orden por números romanos en minúsculas o mayúsculas.



Ejemplo: 1
  1. Enero
  2. Febrero
  3. Marzo

Ejemplo: a
  1. Enero
  2. Febrero
  3. Marzo

Ejemplo: A
  1. Enero
  2. Febrero
  3. Marzo

Ejemplo: i
  1. Enero
  2. Febrero
  3. Marzo

Ejemplo: I
  1. Enero
  2. Febrero
  3. Marzo

Elemento : < li >

La etiqueta HTML li (iniciales de list item, en inglés) es un elemento de lista que demarca cada ítem o punto de una lista, ya sea ordenada (elemento < ol>) o desordenada (elemento < ul>).

En código fuente, su empleo implica la etiqueta < li> al inicio, seguida del texto o descripción correspondiente al punto o ítem de lista, delimitado con la etiqueta de cierre < /li>. Asimismo, requiere la existencia de una lista o menú, lo cual ocurre al anteponer las etiquetas < ul>, < ol> o < menu> y con la escritura de las etiquetas < /ul>, < /ol> o < /menu> como complemento al final.
Ejemplo :

  • Agua
  • Jugo de limón
  • Azúcar.

Elemento : < ul>

Es un signo de grupo que obligatoriamente empieza con < ul> y termina con < /ul>. Lo usas a través de cajas y bloques. Está conformado por uno y más componentes. Dentro del bloque ul, los textos están comprendidos entre < li> y < /li>. La configuración ul se utiliza para exponer listas no ordenadas y sus características descriptivas te ayudarán a aplicar el formato de manera correcta desde el principio.

Elemetos : < h1>, < h2>, < h3>, < h3>, < h5> , < h6>

Las etiquetas para los títulos en HTML son < h1>, < h2>, < h3>, < h3>, < h5> y < h6> para su inicio y en el cierre de esta se utilizan < /h1>, < /h2>, < /h3>, < /h3>, < /h5> y < /h6>. Es indistinto si utilizas minúsculas o mayúsculas, por tanto, es lo mismo < h1> o < H1>.

Ejemplo:

Texto muy grande H1

Texto grande H2

Texto un poco más grande de lo normal H3

Texto normal H4

Texto pequeño H5
Texto muy pequeño H6

Elemento: Style

Con el elemento HTML style, como autor podrás insertar información de hojas de estilo en el documento. Las etiquetas de este elemento , se definen como elementos de cabecera, por lo tanto, su particularidad es que solo pueden ser declaradas en el encabezado del documento.

Etiquetas texto

Elemento: < br>

La etiqueta vacía
está pensada para insertar saltos de línea en un bloque (por ejemplo, para mostrar varios versos de un poema). La separación entre líneas es la misma que dentro de un bloque (normalmente, inferior a la separación entre bloques).

Elemento: < span>

La etiqueta < span> no tiene un significado definido y es la que se debe utilizar cuando las otras etiquetas no son adecuadas. Se suele incluir el atributo class para especificar el significado que se le quiere dar a la etiqueta. Los navegadores no muestran la etiqueta < span> de ninguna forma especial.

Elemento: < em>

a etiqueta < em> está pensada para dar énfasis a su contenido, es decir, para llamar la atención sobre ese contenido. Los navegadores suelen mostrar la etiqueta < em> en cursiva

Elemento: < b>

La etiqueta < b> está pensada para identificar texto sobre el que se quiere llamar la atención, pero sin que eso signifique que es importante, como los términos clave en un resumen, nombre de productos comerciales, etc.

Elemento: < i>

a etiqueta < i> está pensada para identificar texto con un significado especial, como nombres de especies, términos técnicos, expresiones extranjeras, etc. Los navegadores suelen mostrar la etiqueta < i> en cursiva
